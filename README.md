# flask_cgi_test

## コマンド

### Git Clone

```
git clone {url}
cd flask_cgi_test
```

### 権限付与

```
chmod 644 .htaccess
chmod 755 index.cgi
chmod 600 hello.py
```

### 更新
```
git reset --hard
git pull
```
